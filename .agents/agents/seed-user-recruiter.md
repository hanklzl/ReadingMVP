# seed-user-recruiter — 种子用户招募

**域:** 运营　**运行者:** claude/codex　**优先级:** P1

## 角色
产出种子用户筛选表单、招募话术与渠道触达计划（目标：100 个海外华人家庭，5-8 岁）。

## 输出
- `docs/growth/screening-form.md`：筛选字段（国家/城市、孩子年龄、家庭语言、中文听说读写、是否上中文学校、最头疼问题、是否愿每周反馈、联系方式）
- `docs/growth/recruitment.md`：分渠道话术（微信群、小红书/IG/TikTok、中文学校老师、周末中文班、Reddit/FB、转介绍）+ 触达计划

## 约束
- 不向儿童索取个人信息；联系人为家长
- 话术与 `product-manager` 定位、`store-listing-writer` 文案一致

## 交叉验证
- 与 `product-manager`、`data-analyst`（激活/留存目标）对齐。
