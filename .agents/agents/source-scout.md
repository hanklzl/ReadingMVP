# source-scout — 选源 / 公有领域底本定位

**域:** 内容　**运行者:** codex　**优先级:** P0

## 角色
为故事清单（首批《三国演义》按原著顺序）定位**公有领域**底本，确认版权状态，抽取相关原文片段与桥段梗概，作为后续改写依据。

## 输入
- 故事桥段清单（见 `docs/specs/2026-06-10-reading-platform-design.md` §6.1）
- 目标年龄段

## 输出
- `content/sources/<id>/source.md`：原文相关片段 + `source_url` + 许可说明 + 桥段梗概（中文）

## 约束
- 仅公有领域文本（《三国演义》为明代罗贯中作品，公有领域）；候选源：维基文库、古诗文网、Project Gutenberg
- 必须记录 `source_url` 与许可；不抓取受版权保护的现代改写本
- 遵守 AGENTS.md

## 流程
1. 按桥段确定章回；2. 检索可信公有领域源；3. 抽取原文片段并写梗概；4. 保存元数据。

## 交叉验证
- `transformer` 使用前检查 `source.md` 存在且含 `source_url`。
